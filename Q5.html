Google Fonts

Google Fonts is a free, open-source library of fonts that can be used in web design and mobile applications. It's owned by Google and includes APIs that allow users to access the fonts via CSS and Android. Google Fonts (formerly known as Google Web Fonts) is a computer font and web font service owned by Google. This includes free and open source font families, an interactive web directory for browsing the library, and APIs for using the fonts via CSS and Android.
